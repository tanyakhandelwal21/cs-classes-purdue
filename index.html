CS Classes at Purdue

Ratings of Purdue CS Classes


CS 180: Problem Solving And Object-Oriented Programming

CS 180 is a class based on object-oriented programming in Java. It goes from the very basics of Java to developing GUIs in Java. It is easy for the people coming in with some programming experience, however, can be challenging for people who have minimal or no programming experience.

Recommended professor: Dr. Buster Dunsmore


CS 182: Foundations Of Computer Science

CS 182 is a class focussed on Discrete Mathematics. It seems easy but becomes extremely tricky. Take advantages of the PSOs, even though they're optional. The TAs are very helpful and provide great problems.

Recommended professor: Dr. Petros Drineas


CS 240: Programming in C

CS 240 introduces the UNIX command environment and coding in C. Students learn about memory management, working with pointers, and low-level I/O. The good thing about this class is that it involves no theory, all exams and homework assignments focus on writing code to solve a problem.

Recommended professor: Dr. Gustavo Rodriguez-Rivera


CS 250: Computer Architecture

CS 250 delves into hardware. Students learn about how computers are built from the very lowest level. Lab assignments involve making circuits and writing assembly code on a Rasberry Pi. The exams get very tricky.

Recommended professor: Dr. George Adams


CS 251: Data Structures And Algorithms

CS 251 is an extremely crucial course, both, the theoretical and practical components. It involves analysing the space and time complexity of algorithms, and implementing several one-dimensional data structures.

Recommended professor: Dr. Gustavo Rodriguez-Rivera


CS 252: Systems Programming

CS 252 involves low-level programming. It introduces students to writing and managing long programs (>1000 lines of code). The lab assignments are very involved.

Recommended professor: Dr. Buster Dunsmore
